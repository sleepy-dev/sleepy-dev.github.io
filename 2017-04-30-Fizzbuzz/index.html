So let’s start with something simple FizzBuzz.

FizzBuzz is a very basic interview question used to filter out candidates who cannot code. The problem is simple, loop through the natural numbers 1 to 100 and replace each number divisible by 3 with the word “fizz”, each number divisible by 5 with the word “buzz”, replace each number divisible by 3 and 5 with the word “fizzbuzz” and leave the other numbers as is.

A standard simple solution to Fizzbuzz is given below in python:

for i in range(1,101):
    if ((i % 3 == 0) and (i % 5 == 0)):
        print("fizzbuzz")
    elif i % 3 == 0:
        print("fizz")
    elif i % 5 == 0:
        print("buzz")
    else:
        print(i)

The above solution is one that most successful candidates will give when presented with this problem.

However, some people have come up with an “optimized” solution which uses only one if condition as shown below.

output_list = [ "", "", "fizz", "", "buzz",
    "fizz", "", "", "fizz", "buzz",

for i on range(1,101):
	check(output_list[i],i)

The above “optimized” solution might only require one conditional statement, but it is also harder to read and also it has hard coded the output values for 3 and 5, so in my opinion, the first solution is better.
